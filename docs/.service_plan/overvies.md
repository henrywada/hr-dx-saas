# HR-DX SaaS 機能一覧

> 最終更新：2026-05-11

---

## 1. 認証・アカウント管理

| サービス機能名     | 機能詳細                                                    |
| ------------------ | ----------------------------------------------------------- |
| ログイン           | メール＋パスワード認証（Supabase Auth）、テナント別ログイン |
| サインアップ       | 新規テナント登録・従業員アカウント作成                      |
| パスワードリセット | メール送信によるパスワード再設定                            |
| オンボーディング   | 初回ログイン時のセットアップウィザード                      |

---

## 2. 組織・従業員管理（テナント管理者）

| サービス機能名               | 機能詳細                                           |
| ---------------------------- | -------------------------------------------------- |
| 従業員管理                   | 従業員の追加・編集・削除、部署・役職・権限設定     |
| 部署管理                     | 部署（Division）の作成・編集・階層管理             |
| 事業所管理（Establishments） | 事業所単位の登録・管理（ストレスチェック集計単位） |
| サービス割当管理             | テナント従業員へのサービス（機能）アクセス権割当   |
| テナント設定                 | 企業情報・テーマカラー・各種設定管理               |
| お知らせ管理                 | 従業員向けアナウンスメントの作成・配信             |
| マニュアル管理               | 社内マニュアルの登録・閲覧管理                     |

---

## 3. 勤怠管理

| サービス機能名            | 機能詳細                                                             |
| ------------------------- | -------------------------------------------------------------------- |
| QR 出退勤                 | QR コードをスキャンして打刻。従業員端末との Bluetooth ペアリング対応 |
| CSV 出退勤                | CSV ファイルによる勤怠データの一括インポート                         |
| テレワーク勤怠（PC 打刻） | PC ログインをトリガーとした在宅勤務の出退勤記録・承認                |
| 勤怠ダッシュボード        | 打刻状況・出勤率・遅刻・早退のリアルタイム集計表示                   |
| 自己勤怠照会              | 従業員が自分の月次勤怠記録を確認                                     |

---

## 4. 残業・工数管理

| サービス機能名          | 機能詳細                                                         |
| ----------------------- | ---------------------------------------------------------------- |
| 残業申請                | 従業員による残業申請（事前・事後）の登録                         |
| 残業承認                | 上長による残業申請の承認・却下・修正依頼ワークフロー             |
| 月次締め処理（Closure） | 月次勤怠データの集計・ロック・承認フロー                         |
| タイムカード照会        | 締め期間ごとのタイムカード詳細確認・異常検知                     |
| 36 協定分析             | 月間・年間の時間外労働を 36 協定の上限に照らして可視化・アラート |

---

## 5. ストレスチェック・組織健康管理

| サービス機能名                 | 機能詳細                                               |
| ------------------------------ | ------------------------------------------------------ |
| ストレスチェック実施管理       | 法定ストレスチェックの実施設定・回答期間管理・進捗追跡 |
| ストレスチェック回答（従業員） | 従業員が自分でストレスチェックを実施・結果確認         |
| グループ分析                   | 部署・事業所単位でのストレス集計・高ストレス者割合分析 |
| 高ストレス者一覧               | 高ストレス判定者の抽出・管理（個人情報保護ルール準拠） |
| 高ストレスフォローアップ       | 産業医面談の記録・フォローアップ管理                   |
| 行政報告書（Gov Report）       | 厚生労働省向けストレスチェック実施結果報告書の自動生成 |
| パルスストレス                 | 短周期のパルス型ストレス簡易チェック（週次・月次）     |

---

## 6. パルスサーベイ・アンケート

| サービス機能名                   | 機能詳細                                                     |
| -------------------------------- | ------------------------------------------------------------ |
| パルスサーベイ（従業員向け）     | 組織エンゲージメント・コンディション確認の定期アンケート回答 |
| サーベイダッシュボード（管理者） | 回答率・スコアの推移・部署別比較グラフ                       |
| テナント独自アンケート           | 管理者がカスタムアンケートを作成・配信・集計                 |
| 採用候補者パルス                 | 採用プロセス中の候補者満足度サーベイ                         |
| 公開アンケート                   | 認証不要のパブリックアンケートリンク配信                     |

---

## 7. AI 採用支援

| サービス機能名         | 機能詳細                                            |
| ---------------------- | --------------------------------------------------- |
| 求人票管理             | 求人ポジションの登録・編集・ステータス管理          |
| AI 採用支援            | OpenAI を活用した求人票の自動生成・ブラッシュアップ |
| AI 採用ログ            | AI 生成履歴・利用状況のトレーサビリティ確保         |
| 市場分析               | 競合求人・給与相場のデータ分析・レポート            |
| オファー検証           | 提示条件の妥当性チェック・市場比較                  |
| ハローワーク連携       | ハローワーク求人情報との統合・連携機能              |
| 求人インテグレーション | 外部求人媒体との連携設定                            |

## 8. eラーニング

## 13. システム基盤

| サービス機能名                 | 機能詳細                                                       |
| ------------------------------ | -------------------------------------------------------------- |
| マルチテナント RLS             | Supabase Row Level Security によるテナント完全分離             |
| Edge Middleware 認証           | 全ルートの認証チェック・ページビューアクセスログ記録           |
| ロールベースアクセス制御       | 一般従業員 / テナント管理者 / 産業医 / SaaS 管理者の権限分離   |
| ダッシュボード（テナント TOP） | ログイン後ホーム。お知らせ・クイックアクセス・ウィジェット表示 |
| プロフィール・メンバー設定     | 個人プロフィール編集・チームメンバー招待管理                   |
