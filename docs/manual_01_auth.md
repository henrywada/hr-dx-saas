## 1. ログイン手順

1. ブラウザでシステムURLを開きます。  
   システム管理者から案内されたURLを入力してください。

   ![ログイン画面](./images/auth_01_login.png)

2. **メールアドレス** と **パスワード** を入力します。
3. **「ログイン」** ボタンをクリックします。
4. 認証成功後、自動的にポータルトップ（/top）へ遷移します。

> **ポイント：** すでにログイン済みの状態でログインページを開くと、自動的にポータルトップへリダイレクトされます。

---

## 2. パスワードを忘れた場合

1. ログイン画面下部の **「パスワードをお忘れの方」** をクリックします。

   ![パスワード忘れ画面](./images/auth_02_forgot_password.png)

2. 登録済みの **メールアドレス** を入力します。
3. **「再設定メールを送信」** をクリックします。
4. 登録メールアドレスに再設定用リンクが届きます。

> **有効期限：** 再設定リンクは **2週間** 有効です。期限切れの場合は再度手順を行ってください。

---

## 3. パスワードのリセット

1. メールに届いたリンクをクリックし、パスワードリセット画面を開きます。

   ![パスワードリセット画面](./images/auth_03_reset_password.png)

2. **新しいパスワード** を入力し、確認のため同じパスワードをもう一度入力します。
3. **「パスワードを変更する」** ボタンをクリックします。
4. 「変更しました」のメッセージ確認後、ログイン画面から新パスワードでログインします。

---

## 4. 新規サインアップ（企業登録）

本機能は **企業（テナント）の新規契約** 用です。一般社員の方はこの操作は不要です。

1. サインアップページ（/signup）を開きます。

   ![サインアップ画面](./images/auth_04_signup.png)

2. **プランを選択** します（無料 / Pro / Enterprise）。
3. 会社情報・担当者情報を入力します。
4. **Pro プランの場合：** クレジットカード情報を入力して決済します。  
   **Enterprise プランの場合：** 銀行振込の案内が届きます。
5. 登録完了後、サインアップ完了画面（/signup/complete）に遷移し、  
   管理者アカウントのメールアドレスとパスワードが発行されます。

---

## 5. ログアウト

1. 画面右上のアカウントアイコンをクリックします。
2. **「ログアウト」** を選択します。

   ![ログアウト](./images/auth_05_logout.png)

3. ログイン画面に戻ります。

---

## 6. 認証エラーの対処法

| メッセージ | 原因 | 対処 |
|---|---|---|
| 「メールアドレスまたはパスワードが違います」 | 入力誤り | 大文字小文字を確認して再入力 |
| 「ログインが必要です」 | セッション切れ | ログインし直す |
| 「認証エラー：ログインしてください。」 | トークン無効 | 一度ログアウトし再ログイン |
| 再設定リンクが届かない | 迷惑メールフォルダ / メアド誤り | 迷惑メールを確認 or 管理者へ連絡 |
| 「トークンの有効期限が切れています」 | 2週間経過 | 再度「パスワードを忘れた場合」から手順を実行 |
