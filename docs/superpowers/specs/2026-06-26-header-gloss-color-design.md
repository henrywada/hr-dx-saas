# ヘッダー背景色（光沢メタリック化）設計

**日付:** 2026-06-26
**対象:** `/adm`（テナント管理者）・`/saas_adm`（SaaS管理者）の画面ヘッダー背景色

## 背景・目的

`/adm` のヘッダーを「光沢のあるメタリックダークブルー」に、`/saas_adm` のヘッダーを「光沢のあるダークオレンジ」に変更する。`/top`（一般従業員向け）のヘッダーは変更しない。

## 現状

ヘッダーは `src/components/layout/AppHeader.tsx` の単一コンポーネントが `variant`（`'portal' | 'admin' | 'saas'`）prop で背景色を切り替えている（各レイアウトから `AppLayout` 経由で渡される。`/adm` → `variant="admin"`、`/saas_adm` → `variant="saas"`、`/top` → `variant="portal"`）。

現在の背景色（`AppHeader.tsx:50-57`）：
- `portal`：`rgba(255, 255, 255, 0.95)`（変更なし）
- `admin`：`#232a33`（単色）
- `saas`：`#000B00`（単色）

## 変更内容

`AppHeader.tsx:50-57` の `admin`・`saas` 分岐の背景色指定を、単色hexから多段 `linear-gradient` に変更する。

```ts
let bgStyle = 'rgba(255, 255, 255, 0.95)'

if (variant === 'admin') {
  // 光沢のあるメタリックダークブルー
  bgStyle = 'linear-gradient(180deg, #3a4a63 0%, #1c2738 35%, #0f1620 60%, #1c2738 100%)'
} else if (variant === 'saas') {
  // 光沢のあるダークオレンジ（ブランドカラー #FD7601 を暗くしたトーン）
  bgStyle = 'linear-gradient(180deg, #c2530a 0%, #8a3a06 50%, #5c2604 100%)'
}
```

- `saas` の配色はブランドカラー `#FD7601` を起点に明度を落としたトーンを使用し、SaaS管理画面でもブランドの一貫性を保つ（独立した配色は採用しない）。
- 光沢・メタリック表現はCSSグラデーションのみで実装する（`repeating-linear-gradient` 等によるブラッシュメタル質感のオーバーレイは追加しない。YAGNI、保守性を優先）。
- 既存の `boxShadow: 'inset 0 1px 1px rgba(255,255,255,0.2)'`（`headerStyle` 内）がそのまま上部の光沢ハイライトとして機能するため、追加のCSSプロパティは不要。
- ヘッダー高さ（`admin` は `h-12`、`saas`/`portal` は `h-16`）、テキスト色（`text-white`）、アイコンのホバー色（`hover:bg-white/10`）など、背景色以外のスタイルは変更しない。

## 影響範囲

- 変更ファイルは `src/components/layout/AppHeader.tsx` の1ファイルのみ。
- `portal`（`/top`）の表示・ロジックには影響しない。
- ロジック・型・テストへの影響なし（見た目のみの変更）。

## テスト方針

- 自動テストは追加しない（CSS背景色の変更であり、ロジックの単体テスト対象ではない）。
- 手動確認：`/adm` と `/saas_adm` の各画面でヘッダーが意図したグラデーションで表示され、テキスト・アイコンの可読性（コントラスト）が保たれていることを目視確認する。`/top` のヘッダーが変更されていないことも確認する。

## スコープ外（やらないこと）

- ブラッシュメタル質感のテクスチャオーバーレイの追加
- ヘッダー高さ・ボーダー・テキスト色など背景色以外のスタイル変更
- `portal`（`/top`）ヘッダーへの変更
